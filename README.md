# financial-research-AI-Agent
📈 Financial Research AI AgentAn AI-powered stock market research and portfolio management system. This project combines LLM Agents (via Groq & LangChain) with real-time financial data to provide automated technical analysis, sentiment analysis, and portfolio tracking.🚀 FeaturesAI Financial Agent: Uses a ReAct-style agent to answer complex financial queries.Live Portfolio Dashboard: Real-time tracking of stock holdings using yfinance.Interactive Visualizations: High-quality charts for portfolio allocation (Pie) and performance (Bar/Line) using Plotly.Technical Analysis: Built-in tools for RSI, Moving Averages, and MACD indicators.Database Integration: Persistent storage for your portfolio and trade history.📂 Project StructureFileDescriptionapp.pyThe main Streamlit web application. Run this to see the UI.FINTECH_AI_AGENT.ipynbA Jupyter Notebook version used for testing the LangChain Agent and tools.portfolio.dbThe local SQLite database used by the Streamlit app to store your stocks.stock_portfolio_schema.sqlA PostgreSQL schema for scaling the app to a professional cloud database.🛠️ Installation & SetupClone the Repository:Bashgit clone https://github.com/your-username/fintech-ai-agent.git
cd fintech-ai-agent
Install Required Packages:Bashpip install streamlit langchain-groq langgraph yfinance pandas matplotlib plotly textblob ta duckduckgo-search
Set Up Your API Key:Open app.py or your environment variables and add your Groq API Key:Pythonos.environ["GROQ_API_KEY"] = "your_key_here"
Run the Application:Bashstreamlit run app.py
📊 Database ManagementThe application is built to be flexible:Default (SQLite): The app automatically reads from and writes to portfolio.db.Professional (PostgreSQL): If you wish to migrate to a cloud database, use the provided stock_portfolio_schema.sql to initialize your tables. It includes sample data for RELIANCE.NS, AAPL, and BTC-USD.🤖 How the AI WorksThe agent is equipped with a "Toolbox" that allows it to:Fetch Prices: Get real-time data from Yahoo Finance.Search News: Use DuckDuckGo to find the latest market sentiment.Analyze Trends: Calculate technical indicators automatically.📝 LicenseThis project is for educational and research purposes.
